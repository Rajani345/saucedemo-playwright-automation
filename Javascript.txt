Variables in JAVASCRIPT;
===============================
Let, Const and  var //var is not used for latest versions
Var = Global level/Fuction level (Can be redeclared and reinitialized
let = global level/ block level (Can't be redeclared but reinitialized)
Const = global level/ block level (Can't be redeclared and reinitialized)

eg;

var name = 'Rajaesh'     ///Redeclared with same variable and variable name
